# st-finntop

A Bloomberg-style market news terminal built with [Streamlit](https://streamlit.io/) and the [Finnhub API](https://finnhub.io/) built with claude code.

## Features

- **Market News** — browse general, forex, crypto, and M&A headlines with category tabs
- **Company News** — search by ticker symbol for company-specific articles
- **Symbol Lookup** — find ticker symbols by company name or keyword
- Auto-refreshing news feed with configurable timezone display
